# learning-csharp

A project-driven learning lab for C# and .NET. Instead of jumping straight into ASP.NET Core or Unity, the plan here is to combine a knowledge map with a sequence of progressively harder small projects — language fundamentals first, then .NET engineering, async, web APIs, data access, background services, and diagnostics.

## Navigation

- [roadmap.md](roadmap.md) — the knowledge map. What to learn, organized by stage.
- [projects/README.md](projects/README.md) — the action plan. Fifteen projects from CLI tool to performance lab, with recommended order and iteration method.

Each project has its own folder under `projects/` with a self-contained README.
