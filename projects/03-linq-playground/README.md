# Project 3: LINQ Data Query Playground

Build a small in-memory data analysis project with sample domain data.

## Example Domains

- Orders and customers
- Books and authors
- Movies and ratings
- Employees and departments

## Learning Modules

- LINQ method syntax
- Query syntax
- `Where`, `Select`, `GroupBy`, `Join`, `OrderBy`
- Deferred execution
- `IEnumerable<T>` vs materialized collections
- Projection into records

## Completion Criteria

- Includes at least 15 LINQ queries
- Demonstrates filtering, projection, grouping, joining, and aggregation
- Shows at least one deferred execution example
- Explains when `ToList()` is needed
- Has tests for important queries
