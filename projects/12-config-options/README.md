# Project 12: Configuration and Options Demo

Build a small app that demonstrates modern .NET configuration patterns.

## Features

```text
appsettings.json
environment variables
command-line overrides
validated options
multiple environments
```

## Learning Modules

- `Microsoft.Extensions.Configuration`
- Options pattern
- Options validation
- Environment-specific configuration
- Dependency injection
- Logging

## Completion Criteria

- Loads configuration from multiple sources
- Binds configuration into strongly typed options
- Validates options at startup
- Demonstrates environment-specific behavior
- Avoids passing raw configuration dictionaries throughout the app
