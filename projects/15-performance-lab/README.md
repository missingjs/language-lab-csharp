# Project 15: Performance Optimization Lab

Choose a previous project and optimize it with measurement.

## Good Candidates

- Log Analyzer
- File Backup CLI
- Async URL Checker
- Background Job Queue
- Bookmark API

## Learning Modules

- BenchmarkDotNet
- Allocation analysis
- GC behavior
- `Span<T>` and `Memory<T>` basics
- Profiling
- Avoiding premature optimization

## Completion Criteria

- Has a benchmark baseline
- Identifies a specific bottleneck
- Compares before and after results
- Explains why performance changed
- Does not optimize blindly
