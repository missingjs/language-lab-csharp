# Project 9: Minimal Web API

Build a small ASP.NET Core Minimal API.

## Example Endpoints

```text
POST   /bookmarks
GET    /bookmarks
GET    /bookmarks/{id}
DELETE /bookmarks/{id}
```

First version can use in-memory storage.

## Learning Modules

- ASP.NET Core Minimal APIs
- Routing
- Request/response models
- JSON serialization
- Dependency injection
- Validation
- Logging
- HTTP error responses

## Completion Criteria

- Domain logic is not embedded directly in endpoint lambdas
- Request and response models are typed
- Validation errors return clear responses
- Uses dependency injection
- Has endpoint or integration tests
