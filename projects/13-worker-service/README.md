# Project 13: Worker Service

Build a production-style worker service.

## Example Use Cases

- Periodic report generator
- Queue consumer
- Directory watcher
- Scheduled cleanup job

## Learning Modules

- Worker Service template
- `IHostedService`
- `BackgroundService`
- Dependency injection
- Logging
- Configuration
- Graceful shutdown
- Health checks, if applicable

## Completion Criteria

- Runs as a long-lived service
- Handles cancellation correctly
- Logs lifecycle events
- Uses configuration and DI
- Has testable core logic separate from the hosting layer
