# Project 14: Diagnostics and Observability Lab

Add production diagnostics to one of the previous services.

## Learning Modules

- Structured logging
- Metrics
- OpenTelemetry basics
- Health checks
- `dotnet-counters`
- `dotnet-trace`
- Error reporting

## Completion Criteria

- Uses structured logs with relevant context
- Exposes health checks where appropriate
- Emits useful metrics
- Can be inspected with .NET diagnostic tools
- Documents how to debug a common failure scenario
